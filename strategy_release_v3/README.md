# Block E: Momentum-Trend Strategy v3 (Single Set + ML Overlay)

Это **v3** версия торговой стратегии Momentum-Trend для инструмента RTSF.
Она возвращается к корням "снайперского" подхода из v1 (использование ОДНОГО лучшего набора параметров для LONG и SHORT), но с добавлением **AI-скоринга (ML Overlay) на базе XGBoost**, разработанного в v2.

## 🚀 Зачем нужна v3? (Отличия от v1 и v2)

- **Проблема v1 (Single Set):** Огромная абсолютная прибыль (+445%), но с риском просадок до 16-20%.
- **Проблема v2 (Ensemble):** Идеально гладкая кривая эквити (просадка 5-10%), но из-за дробления капитала (математическое округление `floor`) и комиссий ансамбля, абсолютная прибыль сильно падает.

**Решение в v3:** Мы берем ОДИН вылизанный набор параметров из v1 (который дает сделки с огромным математическим ожиданием >500 пунктов), используем капитал 5M целиком (без потери контрактов на дроблении) и ставим поверх него XGBoost фильтр для защиты от "пилы" и плохих рыночных режимов.

## ⚠️ Важное замечание по результатам v3

В ходе тестов v3 мы реализовали подход **Joint Optimization (Совместная оптимизация)**: мы перебрали 50 лучших наборов параметров из GPU-прогона и для каждого обучили свою XGBoost модель. Цель была найти такую комбинацию (Параметры + ИИ), которая даст максимальную прибыль после фильтрации.

### Результаты Joint Optimization (OOS 2019-2026)

Лучшая найденная комбинация дала следующие метрики:
- **Базовая стратегия (выбранная для ИИ):** 344 сделки, Profit Factor 1.61, Max DD 14.06%
- **После ИИ-фильтра (Порог 0.5, Плечо x2.0):** 156 сделок, Profit Factor 1.50, Max DD 9.31%
- **С динамическим плечом (ML Filtered + Leverage x4.0):** Net Profit 5,396,785, Max DD 15.15%

**Вывод по v3 (Joint Optimization):**
Совместная оптимизация частично сработала! Мы смогли найти набор параметров, который генерирует достаточно много сделок (~1000 за всю историю), чтобы XGBoost смог найти закономерности. 
- Просадка снизилась с 14.06% до 9.31%.
- Повышение плеча до x4.0 позволило получить феноменальную абсолютную прибыль (почти 5.4 млн на OOS) при просадке всего 15.15%.

Тем не менее, стоит отметить, что Ансамбль (v2) всё еще предоставляет более гладкую кривую эквити за счет диверсификации по таймфреймам. Версия v3 подходит для трейдеров, которые готовы мириться с чуть более высокой волатильностью баланса ради максимизации абсолютной прибыли.
